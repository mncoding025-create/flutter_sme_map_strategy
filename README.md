## 📌 เกี่ยวกับโปรเจกต์ (About The Project)
**SME Strategy Map** คือแอปพลิเคชันสำหรับการวิเคราะห์ทำเลและยุทธศาสตร์ทางธุรกิจ (Business Intelligence) ออกแบบมาเพื่อช่วยผู้ประกอบการประเมินศักยภาพของพื้นที่โดยรอบ ไม่ว่าจะเป็นกลุ่มลูกค้า คู่แข่ง หน้าร้านของตนเอง หรือซัพพลายเออร์ พร้อมแสดงผลข้อมูลในรูปแบบแผนที่และแดชบอร์ดสถิติที่เข้าใจง่าย เพื่อสนับสนุนการตัดสินใจทางธุรกิจที่แม่นยำยิ่งขึ้น

## ✨ ฟีเจอร์หลัก (Key Features)
* **📍 Interactive Map:** ระบบแผนที่ปักหมุดอัจฉริยะ พร้อมจัดหมวดหมู่สีตามประเภท (ลูกค้า, หน้าร้านเรา, คู่แข่ง, ซัพพลายเออร์)
* **📊 Smart Dashboard:** หน้าสรุปดัชนีชี้วัดหลัก (KPIs) และกราฟแสดงสัดส่วนกลุ่มอายุลูกค้าเป้าหมาย
* **📝 Data Management:** ระบบเพิ่ม แก้ไข และลบข้อมูลพิกัดยุทธศาสตร์ได้แบบ Real-time
* **🗺️ Navigation:** เชื่อมต่อพิกัดเข้ากับแอปพลิเคชัน Google Maps เพื่อนำทางได้อย่างรวดเร็ว
* **☁️ Cloud Sync:** ซิงค์ข้อมูลทั้งหมดอย่างปลอดภัยด้วยฐานข้อมูล Supabase

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Frontend:** Flutter (Dart)
* **Backend & Database:** Supabase Cloud
* **Mapping Service:** Google Maps Platform
